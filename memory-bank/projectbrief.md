# Project Brief

## Project
Agentic CBA Indicators

## Goal
Provide a CLI chatbot that queries weather, climate, socio-economic data, and CBA ME Indicators using Strands Agents with multi-provider model support.

## Scope
- CLI runtime (agent creation, configuration, prompts)
- Tooling integrations for external APIs
- Knowledge base ingestion pipelines (Excel, use cases)
- Configuration and paths handling

## Success Criteria
- Reliable first-run experience with bundled configuration
- Robust, consistent HTTP interactions across tools
- Deterministic, high-integrity ingestion with explicit failure handling
- Centralized shared utilities to reduce duplication
- Adequate automated coverage for core functionality
